### Tone Attributes

| Attribute | What It Means |
|-----------|---------------|
| Calm confidence | We've seen this before. We know what works. |
| Operational logic | Clear cause → effect reasoning |
| Slight contrarianism | "While others chase trends..." |
| Specific and grounded | Exact numbers, concrete mechanisms |
| Respectful of experience | Acknowledge what they already know |

### Target ICP (Revised)

**Primary:** Experienced ecommerce operators (45-65)
- $500K-$10M annual revenue
- Business-literate but not "AI-native"
- Skeptical of hype, responsive to logic
- Under margin pressure, seeking efficiency

**Not For:**
- Brand new stores without traffic
- DIY operators who want raw data
- Enterprise with complex compliance needs
- Anyone looking for magic

## Implementation Notes

### What to Keep

- Clean visual design (already good)
- Basic page structure
- Integration logos
- Pricing model (just display it transparently)

### What to Remove

- Monster/Godzilla imagery and metaphors
- Emojis in professional copy
- Fake scarcity ("Only 8 spots left")
- Tribal language ("herd", "elite")
- Aggressive CTAs ("Crush leaks")
- Competitor attack copy

### What to Add

- Transparent pricing on page
- Explicit "who this is for / not for" section
- FAQ section addressing common objections
- Simple process explanation (4 steps)
- Risk reducers throughout (trial, no card, cancel anytime)

## Messaging Hierarchy

Every page should answer these questions in order:

1. **What problem exists?** → Fragmented data architecture
2. **Why now?** → iOS changes, rising costs, margin pressure
3. **Why don't existing tools solve it?** → Dashboards without direction
4. **How does Growzilla solve it?** → Connects, analyzes, recommends
5. **What does using it look like?** → 90 seconds → 24 hours → weekly
6. **What results to expect?** → 23% median CVR improvement
7. **Why is switching safe?** → Trial, no card, cancel anytime

---

## Value Equation

All copy reinforces:

```
          (Dream Outcome × Perceived Likelihood)
Value = ─────────────────────────────────────────
           (Time Delay × Effort & Sacrifice)
```

- **Increase numerator:** Specific results, clear mechanism, risk reversal
- **Decrease denominator:** 90-second install, no learning curve, cancel anytime
